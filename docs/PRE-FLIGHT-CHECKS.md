# Pre-Flight Checks

Use these commands to catch syntax errors **before** running the app:

## Quick Commands

```bash
# Check for syntax errors (await, duplicates, etc.)
npm run test:syntax

# Check for case-sensitive import issues
npm run test:imports

# Run all pre-flight checks
npm run lint

# Full test suite
npm run test:all
```
