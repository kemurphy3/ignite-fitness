**Impact**: Could cause conflicts and unexpected behavior

```javascript
function validateInput(value, type, min, max) {
    if (!value) return false;
    if (type === 'number') {
        const num = parseFloat(value);
        return !isNaN(num) && num >= min && num <= max;
    }
    return true;
}
```
